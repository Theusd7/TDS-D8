=======================
COMANDOS SQL - LISTA COMPLETA
=======================

1. SELECT – Consulta dados
Busca dados em uma ou mais tabelas.
Exemplo:
SELECT nome, idade FROM clientes;

2. INSERT INTO – Insere dados
Insere um novo registro (linha) na tabela.
Exemplo:
INSERT INTO clientes (nome, idade) VALUES ('João', 30);

3. UPDATE – Atualiza dados existentes
Modifica registros existentes.
Exemplo:
UPDATE clientes SET idade = 31 WHERE nome = 'João';

4. DELETE – Remove dados
Apaga registros da tabela.
Exemplo:
DELETE FROM clientes WHERE nome = 'João';

5. CREATE TABLE – Cria uma nova tabela
Define uma nova estrutura de tabela no banco.
Exemplo:
CREATE TABLE clientes (
  id INT PRIMARY KEY,
  nome VARCHAR(100),
  idade INT
);

6. DROP TABLE – Exclui uma tabela
Apaga a tabela e todos os seus dados.
Exemplo:
DROP TABLE clientes;

7. ALTER TABLE – Altera a estrutura da tabela
Adiciona, remove ou altera colunas.
Exemplo:
ALTER TABLE clientes ADD email VARCHAR(150);

8. WHERE – Filtro de dados
Restringe os registros retornados ou afetados.
Exemplo:
SELECT * FROM clientes WHERE idade > 25;

9. ORDER BY – Ordena os resultados
Ordena os dados por uma ou mais colunas.
Exemplo:
SELECT * FROM clientes ORDER BY nome ASC;

10. GROUP BY – Agrupa registros
Agrupa dados para usar com funções agregadas.
Exemplo:
SELECT cidade, COUNT(*) FROM clientes GROUP BY cidade;

11. HAVING – Filtro após agrupamento
Filtra resultados depois do GROUP BY.
Exemplo:
SELECT cidade, COUNT(*) FROM clientes GROUP BY cidade HAVING COUNT(*) > 10;

12. JOIN – Junta dados de tabelas diferentes
Relaciona registros entre tabelas.
Exemplo:
SELECT pedidos.id, clientes.nome
FROM pedidos
JOIN clientes ON pedidos.cliente_id = clientes.id;

13. DISTINCT – Remove duplicados
Retorna apenas valores únicos.
Exemplo:
SELECT DISTINCT cidade FROM clientes;

14. LIMIT / TOP – Limita o número de resultados
Restringe a quantidade de linhas retornadas.
Exemplo (MySQL):
SELECT * FROM clientes LIMIT 5;

15. LIKE – Busca com padrões (strings)
Busca parecida com curinga (%).
Exemplo:
SELECT * FROM clientes WHERE nome LIKE 'J%';

16. IN – Verifica múltiplos valores
Filtra registros que estejam dentro de um conjunto.
Exemplo:
SELECT * FROM clientes WHERE cidade IN ('São Paulo', 'Rio de Janeiro');

17. IS NULL / IS NOT NULL – Verifica valores nulos
Exemplo:
SELECT * FROM clientes WHERE email IS NULL;

18. CASE – Condicional (tipo IF/ELSE)
Exemplo:
SELECT nome,
       CASE 
         WHEN idade >= 18 THEN 'Adulto'
         ELSE 'Menor'
       END AS classificacao
FROM clientes;
